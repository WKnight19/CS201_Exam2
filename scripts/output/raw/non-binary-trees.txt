N-ary Tree
• A Nonbinary Tree, also known as an N-ary Tree, is a tree
data structure in which each node can have zero or more
child nodes, up to a maximum of 𝑁 children.
• An N-ary Tree is defined as a rooted tree in which each node
has at most 𝑁 children.
o When 𝑁 = 2, the structure becomes a binary tree.
o When 𝑁 > 2, it is called a non-binary or N-ary tree.

Level-order Traversal
void levelOrder(Tree T) {
Queue Q(); // FIFO
Q.enqueue(T.root);
while (not Q.isEmpty()) {
Node p = Q.dequeue();
visit(p);
for (Node c = p.leftmostChild; c != null; c = c.rightSibling)
Q.enqueue(c);
}
}

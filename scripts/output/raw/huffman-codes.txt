Huffman Codes
• Huffman codes are variable-length binary codes assigned to
symbols based on their frequencies.
• Symbols that occur more often are given shorter codes,
while less frequent symbols are given longer codes,
resulting in efficient and lossless data compression.
• These codes are generated using the Huffman coding
algorithm, which builds a binary tree that minimizes the
total number of bits required to represent the data.

Greedy Huffman Algorithm (cont.)
1. Construct a Huffman tree (binary tree)
H = new Heap(); // Min-heap
for j = 1 to n
H.insert( new Leaf(F[j], A[j]) ); // F[j] is key
while (H.size() >= 2) {
left = H.removeMin();
right = H.removeMin();
sum = left.key + right.key;
H.insert(new Node(sum, left, right)); // sum is key
}
